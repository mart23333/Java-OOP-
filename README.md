# Java-OOP Quiz Game 
-Console based java quiz game application designed to help students test their knowlegde of java  OOP concepts.
this project demostrates OOP principles, multithreading, file handling and user interaction 
 # Features 
- Different question types (boolean, mutiple choice questions)
-Question shuffling 
-20 seconds timer for each question 
-Player profile ( player must enter name before starting )
-High score tracking saved to text file 
-Score display at the end of the quiz 
# OOP Concepts Used 
-Inheritance: question subclass extends the abstract Question class
-Polymorphism : each question types overides ask()
-Encapsulation : it protects fields like correct answers 
-Interface : IQuestionGenerator to ensure flexible question creation
-Multithreading : Timethread to handle countdowns 
-File Handling : saves and loads high scores

# How to run 
-Open the project in any java IDE 
-Run Main.java class 
-Enter name and start the quiz 
-Answer the Questions before time runs out 
-See your score and high score
